# InforcapHouse — Modelo de Negocio

> Marco: **Lean Canvas**. Describe por qué existe el proyecto y cómo genera valor.
> **Última actualización**: 2026-07-02

## 1. Problema

- InforcapHouse publica sus inmuebles solo en grupos de Facebook e Instagram, sin una presencia web propia, confiable y estable.
- La información de los inmuebles está dispersa y no es fácil de consultar ni filtrar para los interesados.
- **Alternativas actuales**: publicaciones en redes sociales (Facebook/Instagram) y portales inmobiliarios de terceros.

## 2. Segmentos de cliente

| Segmento             | Descripción                                   | Notas          |
| -------------------- | --------------------------------------------- | -------------- |
| Interesados en inmuebles | Personas que buscan comprar o arrendar propiedades | Público objetivo principal |
| Equipo InforcapHouse | Administradores que publican y gestionan inmuebles | Rol `admin`    |

- **Cliente ideal (ICP)**: comprador/arrendatario que busca inmuebles catalogados de forma clara y confiable en un sitio propio.
- **Early adopters**: la audiencia actual de InforcapHouse en Facebook e Instagram.

## 3. Propuesta de valor única

- Un catálogo web propio de inmuebles, confiable y siempre disponible, sin depender de las redes sociales.
- **Concepto de alto nivel**: "el escaparate online de InforcapHouse para sus inmuebles".

## 4. Solución (el producto)

| Módulo / Capacidad            | Qué resuelve                                        |
| ----------------------------- | --------------------------------------------------- |
| Catálogo de inmuebles         | Visualizar propiedades con tipo, oferta, precio, área, características y fotos |
| Autenticación y gestión (admin) | Publicar, editar y eliminar inmuebles de forma segura |
| Formulario de contacto        | Que los interesados se comuniquen con la empresa    |
| Páginas estáticas             | Inicio y términos legales                            |

## 5. Canales

- Sitio web propio (este proyecto) como canal principal.
- Redes sociales existentes (Facebook, Instagram) para dirigir tráfico al sitio.

## 6. Modelo de ingresos

Este proyecto es el **sitio de catálogo (MVP)**, no un producto SaaS con planes de suscripción. Los ingresos provienen del **negocio inmobiliario de InforcapHouse** (venta/arriendo de inmuebles), no del software en sí.

| Concepto                 | Detalle                                             |
| ------------------------ | --------------------------------------------------- |
| Fuente de ingresos       | Comisión/margen por venta o arriendo de inmuebles   |
| Rol del sitio            | Captación y presentación de inmuebles / leads       |
| Planes de suscripción    | No aplica en esta etapa                             |
