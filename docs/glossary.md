# Glosario

Vocabulario compartido del dominio y del proyecto. Mantén las definiciones cortas
y sin ambigüedad para que todo el equipo use los términos de la misma forma.

| Término       | Definición                                                                 |
| ------------- | -------------------------------------------------------------------------- |
| Inmueble (Property) | Propiedad publicada en el catálogo: tiene tipo, oferta, descripción, área, precio, características y fotos. |
| Tipo de oferta (TypeOffer) | Clasificación de la transacción del inmueble (p. ej. venta, arriendo). |
| Tipo de inmueble (TypeProperty) | Clasificación del inmueble (p. ej. casa, departamento, local). |
| Característica (Feature) | Atributo seleccionable de un inmueble (p. ej. piscina, estacionamiento); relación muchos-a-muchos. |
| Admin         | Usuario con rol `admin`, único que puede gestionar (crear/editar/eliminar) inmuebles. |
| Usuario regular | Usuario con rol `regular`: navega el catálogo y usa el formulario de contacto. |

> Convención: ordena los términos alfabéticamente y enlaza al documento donde se
> detalla cada concepto cuando aplique.
